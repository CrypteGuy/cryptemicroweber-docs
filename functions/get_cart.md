## get_cart

get_cart - gets the user shopping cart

    $cart_items = get_cart();

### Parameters and fields

You can pass parameters as string or as array.


| optional parameter  |  usage  |
|---|---|
|  order_completed | `get_cart("order_completed=1");`  |
|  order_id | `get_cart("order_id=5");`  |
