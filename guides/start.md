# Getting Started


### Server Requirements
* PHP >= 8.1
* PHP Extensions:
  * intl
  * zip
  * bcmath
  * curl
  * mb_string
  * openssl
  * pdo_mysql
  * pdo_sqlite
  * sodium
  * gd2
  * lib-xml with DOM support
* mod_rewrite if using Apache


### Recommended Software
* Apache >= 2.4
* MySQL >= 5.0
* [Composer](https://getcomposer.org/)

### Supported browsers
* Google Chrome
* Mozilla Firefox
* Webkit-based browsers
